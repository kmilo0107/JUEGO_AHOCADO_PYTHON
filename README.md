# 🎮 Juego del Ahorcado en Python

Juego de ahorcado en consola con nombres colombianos.

## Cómo jugar
- El juego elige una palabra al azar
- Tienes 3 intentos para adivinar letra por letra
- Si completas la palabra, ¡ganas!

## Cómo ejecutar
python ahorcado.py

## Tecnologías
- Python 3
- Librerías: random, os
